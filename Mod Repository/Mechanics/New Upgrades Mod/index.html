    New Upgrades Mod

    Description

    In every nine standard H3 town, you have opportunity to upgrade creatures once more. Mod is based at ERA's Third Upgrades Mod v2.17.0. Compatibility patches with Tides of War and HotA Conflux changes will be added later.

    Version

    1.0.2

    Author

    VMaiko, avatar
